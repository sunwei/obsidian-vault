### 1. Installation & Setup

**Q: Where can I find and install the Friday plugin in Obsidian?**

**A:** Go to **Settings → Community Plugins → Browse**, search for **Friday**, then click **Install** and **Enable**.

---

### 2. Publishing Notes

**Q: How do I publish a single note or an entire folder as a website?**

**A:** Right-click on a note or folder and select **Publish to Web**. Friday will build the site and show a local preview.

---

### 3. Themes

**Q: What’s the difference between the _Book_ and _Obsidian Book_ theme?**

**A:**

- **Book:** Uses standard Markdown, works across platforms
- **Obsidian Book:** Supports Obsidian Markdown features and plugins

**Q: How can I switch themes without editing files manually?**

**A:** Open the Friday panel → click **Change Theme** → choose the theme

**Q: Which Obsidian plugins are supported when publishing with Friday?**

**A:** Plugins that output content through **code blocks**, such as **Timeline** or **Dataview**, are supported.

---

### 4. Features

**Q: Does Friday support publishing multilingual websites?**

**A:** Yes. Refer to this Demo

**Q: Is Friday available on both desktop (Windows) and mobile (iOS)?**

**A:** Yes.

**Q: Will my site automatically adjust for mobile devices?**

**A:** Yes, Friday websites are responsive by default.

**Q: How can I embed videos, tweets, or comments in my notes?**

**A:** Use the built-in **Shortcodes**. Insert the shortcode syntax directly in your Markdown file.

**Q: Can different pages of my site use different themes?**

**A:** Yes. Each note can use its own **theme**.

---

### 5. Privacy & Data

**Q: Does publishing with Friday put my private notes at risk?**

**A:** No. All processing happens locally on your device. Your content is never uploaded to external servers unless you deploy it yourself.
